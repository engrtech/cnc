To make an offset:
- Switch to MDI mode
- EXTEN button
- Next Item
- Sizing Setting
- Offset
(Increate to make smaller)

To change Dress Points:
- Switch to MDI mode
- CHANGE OVER button
- Setting Data
- Next
- Turn key to make program changes

To Run Dress Cycle
- Switch to Manual Mode
- Advance center
- Close door
- Index Part Rotation
- Start Dress Cycle
